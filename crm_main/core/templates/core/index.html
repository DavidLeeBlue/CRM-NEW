Welcome to Django CRM

This CRM system is developed using the Django framework in Python.
It leverages Bootstrap for the front-end and MySQL as its database.
The system facilitates customer relationship management, allowing businesses to efficiently manage interactions with their clients, and track leads.

Basic Functions

This CRM system includes the following basic functions:

  • Leads: Manage potential clients or sales leads.
  • Clients: Manage client information.
  • Products: Manage products.
  • Orders: Manage orders.
  • User Profiles: Manage user profiles, including login and logout.
  • Dashboard: The main interface for users after they log in.
